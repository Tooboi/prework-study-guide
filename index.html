Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add changed files in our working directory to the staging area of Git
  • add -A: command to commit all files in the working branch
  • git commit -m "added code to HTML file": commit to repository
  • -m: flag to associate a message with our commit
  • git pull origin main: sync our local branch with the remote branch in GitHub
  • git push origin feature/yourfeature: push up changes to then be approved and merged on github
  • git checkout main: go to main folder
  • git pull origin main: pull remote to local branch
