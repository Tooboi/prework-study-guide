Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add changed files in our working directory to the staging area of Git
  • add -A: command to commit all files in the working branch
  • git commit -m "added code to HTML file": commit to repository
  • -m: flag to associate a message with our commit
